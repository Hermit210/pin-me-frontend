##  Decentralized Deployment (PinMe)
This frontend is deployed using **PinMe**, which pins the UI to **IPFS** and serves it via the **eth.limo ENS gateway**.
This ensures:
- Tamper-proof delivery  
- Censorship resistance  
- Verifiable content-hash integrity  
- Decentralized availability
